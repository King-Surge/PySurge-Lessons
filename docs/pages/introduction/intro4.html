Python Comments

You've successfully completed your first program and would like to make notes explaining what your code does. However, you do not want your notes to be displayed in the console. Comments allow for you to this.

Code written with Python can be explained through comments. Comments can be used to enhance the readability of the code. When testing code, comments can be used to prevent execution.

There are three types of comments: single-line, multi-line, and docstring comments. The syntax of comments varies depending on the type.

Single-line comments begin with the “#” character. Anything that is written in a single line after ‘#’ is considered as a comment. There are two ways of using single-line comments in Python. You can use it before the code or next to the code.

In the code below only line 2 will be executed up to where the comment starts 

          
            # This is my first single-line comment
            print("Hello, World!") # look I can be here too
            # print("this will not be printed")

Output:

          
            Hello, World!

Python does not support multi-line comments. However, there are multiple ways to overcome this issue. This method is technically multi-line single-line comments. The first way is by using # at the beginning of each line of the comment.

You can also use multi-line strings for commenting. To do this, use either a ‘’ or “” quotation marks three times.

The next way is by using string literals but not assigning them to any variables (Learn further in the Course). If you do not assign a string literal to a variable, the Python interpreter ignores it. Use this to your advantage to write multi-line comments. You can either use a single (‘’) quotation or double (“”) quotation.

In the code below you will see examples of these 

          
            # These
            # are multi-line
            # single-line
            # comments


            """
            This is method 
            number 2
            """

            "This is method"
            "number 3"
