First Program

In Python your first program may feel like one of your gratest achievements. Though very simple, Everyone enjoys doing their first "Hello, World!" program.

In a "Hello, World!" program all you would be doing is printing hello world to the console. You may have seen an example of this on the home page.

Python files usually have the extension "*.py".

"*" represents the name of the file 

          
            print("Hello, World!")

Output:

          
            Hello, World!
